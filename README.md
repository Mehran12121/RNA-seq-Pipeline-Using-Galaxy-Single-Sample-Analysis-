🔬 RNA-seq Analysis Using Galaxy

This project demonstrates a complete RNA-seq analysis workflow performed using the Galaxy platform.

⚙️ Tools Used
FastQC
Trimmomatic
Bowtie2
featureCounts
🔗 Workflow

FASTQ → Quality Control → Trimming → Alignment → Gene Counting

📊 Results
Raw sequencing data was assessed for quality using FastQC
Low-quality reads were removed using Trimmomatic
Reads were aligned to the mouse reference genome (mm10)
Gene expression counts were obtained using featureCounts
🧬 Dataset

Source: NCBI SRA
Accession: SRR1552444
